3가지 이상의 기능/요소 등의 처리를 반복 수행할 경우에는 매번 하나씩 만드는 것이 아닌 반복문을 사용하여 처리

  • while : 여러개의 반복 수행을 처리하는 문법 while(){}
    다른 반복문에 비해 속도가 느리고, 특정 조건 수행 후 반복 종료 기능을 미작성하는 경우가 있어 권장 X
  • do-while: while의 확장형. 조건을 파악했을 때 기본 반복문은 처음부터 수행하지 않을 수 있지만
    do-while은 최소 1번은 수행. do{} while()
  • for : 가장 많이 사용되는 반복문의 기본 형태.
    최초의 값/조건/이후 수행처리에 대한 내용을 한군데에 모두 작성하여 반복문의 수행 속도를 개선
  • for-in : for의 변형된 형태로 배열에서 많이 사용한다고 알려져 있지만 배열에서는 지양. for(){}
    오히려 객체에서 사용하는 기법. for(in){}
  • forEach : for의 변형된 형태로 배열의 반복으로 사용되는 기법. 배열 요소를 각각 수행한다는 의미. [].forEach(function(data,index){})
