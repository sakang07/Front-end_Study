웹

데이터 타입(형타입)

  • 숫자: 실수, 정수 구분 없이 모두 숫자
  • 문자열: 반드시 '' 내부에 글자를 작성. 숫자를 적더라도 '' 내부에 있다면 문자열
  • 논리형: true, false
  • undefined: 값이 지정되지 않은 상태.
  • null: 값이 없다.

특수형

  • 객체(배열/함수)
  • 배열: html 문서에서 말하는 ol처럼 여러 개의 목록으로 [] 안에 내용을 나열
  • 객체: 배열처럼 나열하기는 하지만 순서는 없고, 해당하는 목록에 대한 정의를 하는 것. {} 내부에 여러 개의 내용을 담아서 그룹화 시킨다.
  • 함수: 바로 실행시키거나 내용을 담는 기능이라기보다는 어떠한 행동을 처리하기 위해 만든 묶음
    (선언식, 표현식, 즉시실행함수)

다른 프로그래밍 언어와 다른 점 중 대표적 하나는 객체 내부에 배열/함수를 담을 수 있다는 것.
