양방향 전송 방식

검색, 로그인, 회원가입, 구매, 게시판...

  1. 검색하기 위해 필요한 단어/문장을 작성(글을 작성할 수 있는 영역)
  2. 전송 버튼을 클릭(a는 data를 보내는 능력이 없다. data 전송 버튼)
  3. 데이터를 전송(어디로 보낼지, method: 공개(GET)/비공개(POST) 여부)
  4. 전송받은 곳에서 무언가 수행(backend)
  5. 수행에 대한 결과를 피드백
  6. 피드백 받은 결과를 도출
